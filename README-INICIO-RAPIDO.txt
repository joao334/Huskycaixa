COMO USAR
1. Extraia a pasta.
2. Abra o arquivo index.html.
3. Se o Supabase estiver indisponível, o sistema entra em modo local.
4. Primeiro acesso local padrão:
   E-mail: admin@husky.com
   Senha: 123456

ATALHOS
- Ctrl + K = abrir central de atalhos
- botão flutuante “Atalhos” = abrir navegação rápida
- botão “Tema” = alternar entre claro e escuro

OBSERVAÇÃO
Se você quiser usar autenticação em nuvem e sincronização completa,
mantenha os arquivos js/env.js e js/supabase-init.js configurados.
