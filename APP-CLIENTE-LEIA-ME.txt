APP DO CLIENTE • HUSKY CONFEITARIA

ARQUIVOS PRINCIPAIS
- app-cliente.html = tela de pedido para o cliente no celular
- pedidos-online.html = painel interno para receber/importar pedidos
- SUPABASE-PEDIDOS-CLIENTE.sql = estrutura necessária no Supabase

COMO ATIVAR CERTO
1) Execute o arquivo SUPABASE-PEDIDOS-CLIENTE.sql no SQL Editor do Supabase.
2) Entre no sistema interno.
3) Abra a página "Pedidos online".
4) Clique em "Publicar catálogo".

O QUE MUDOU NESTA VERSÃO
- visual da tela do cliente mais bonito e mais profissional
- catálogo com fallback público pelo Storage
- mensagens de erro mais claras
- carrinho flutuante e experiência melhor no celular

SE APARECER "CATÁLOGO AINDA NÃO CONFIGURADO"
- execute o SUPABASE-PEDIDOS-CLIENTE.sql
- depois publique o catálogo outra vez

SE O CATÁLOGO ABRIR, MAS O PEDIDO NÃO ENVIAR
- a tabela customer_orders ainda não foi criada ou liberada
- execute novamente o SUPABASE-PEDIDOS-CLIENTE.sql

DICA
Mesmo que a tabela pública do catálogo falhe, a nova versão tenta carregar o catálogo publicado em arquivo público do bucket husky-files.
